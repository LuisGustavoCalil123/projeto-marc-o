Git & GitHub

Sistema de Gerenciamento e Controle de Versão

O que é Git?

Git é um sistema de controle de versão distribuído utilizado para gerenciar alterações em projetos de software.

O que é GitHub?

GitHub é uma plataforma online utilizada para armazenar repositórios Git e colaborar em projetos.

Branches

Branches permitem desenvolver funcionalidades separadamente sem afetar a branch principal. 

main
 ├── feature-darkmode
 ├── feature-readme
 └── hotfix-layout

Commits

Commits registram alterações realizadas no projeto durante o desenvolvimento.

Pull Requests

Pull Requests são utilizados para revisar e aprovar alterações antes do merge entre branches.
