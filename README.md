# Sistema de Controle de Versão

Projeto desenvolvido para a disciplina de Gerência e Configuração de Controle de Versão.

## Objetivo

Demonstrar na prática o uso de Git e GitHub utilizando:
- commits;
- branches;
- pull requests;
- issues;
- versionamento.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Explicação sobre Git e GitHub
- Demonstração visual de branches
- Interface responsiva
- Dark Mode
- Estrutura organizada para versionamento
